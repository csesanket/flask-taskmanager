# Task Manager - Flask Docker CI/CD Project

A simple, professional to-do/task management app powered by Python Flask, Docker, SQLite, and automated CI/CD pipelines on AWS EC2 with Jenkins.

---

## Features

- Add, edit, and delete tasks
- Clean UI with Bootstrap (or your chosen frontend)
- Persistent storage with SQLite
- Packaged with Docker for easy deployment
- Automated deploys using Jenkins
